# Welcome to Remix!

- [Remix Docs](https://remix.run/docs)

## Architect Setup

```sh
yarn install
yarn postinstall
yarn build
```

## Development

```sh
yarn start
```

Open up [http://localhost:3333](http://localhost:3333) and you should be ready to go!

## Deploying

```sh
make deploy
```

You're in business!
